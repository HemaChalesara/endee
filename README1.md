# AI RAG System using Endee

## Overview
This project implements a Retrieval Augmented Generation (RAG) system using Endee concepts.

## Features
- Semantic Search
- Vector Database
- AI-based Question Answering

## Tech Stack
- Python
- Sentence Transformers
- NumPy
- Endee (concept-based implementation)

## System Design
User Query → Embedding → Vector Search → Context Retrieval → Answer Generation

## Setup
pip install sentence-transformers numpy

Run the notebook.

## Note
Endee repository was forked as required. Since it is not pip-installable, a custom implementation was used.
